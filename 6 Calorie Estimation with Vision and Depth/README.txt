1. Please download the dataset from https://www.kaggle.com/competitions/comp-90086-nutrition-5-k/

2. Please unzip the Nutrition5k data and save it in the same address as the ipynb code file.

3. The Nutrition5K data should be saved in the following structure:
Nutrition5K ---test----color
     |           |-----depth_color
     |           |-----depth_raw
     |
     |---------train---color
                 |-----depth_color
                 |-----depth_raw

4. Run the ipynb file block by block. (It's faster if you have an NVIDIA GPU since we're using PyTorch)
