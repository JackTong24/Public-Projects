Some projects I've done.

1. Tetress AI Solver
An AI program that solves Tetress – a fun strategy game similar to Tetris, played on an 11×11 board that wraps around like a donut.

What it does
As the red player, it finds the shortest sequence of tetromino moves needed to remove a target blue piece from the board.

Why it matters
Shows real AI problem-solving using smart search techniques
Handles tricky wrapping-board logic efficiently
Clean, well-organised Python code with good documentation

Built for the Artificial Intelligence course at the University of Melbourne. Includes the full technical report and test cases.

Technologies: Python, A* Search
