# Public-Projects
Some projects I've done.

1. Tetress AI Solver
An AI program that solves Tetress – a fun strategy game similar to Tetris, played on an 11×11 board that wraps around like a donut.
Technologies: Python, A* Search

2. Tetress AI Game Agent
An intelligent AI agent for Tetress – a two-player strategy game on an 11×11 toroidal board (like a donut-shaped Tetris battlefield). Players take turns placing tetrominoes until one can’t move or the board fills up.
Technologies: Python, Minimax, Alpha-Beta Pruning, Lookahead Search, Performance Optimization
